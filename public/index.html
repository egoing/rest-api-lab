REST API 실험실

준비

  1. README.md파일을 참고해서 설정합니다.
  2. node.js 설치
  3. json-server 설치
    npm i -g json-server
  4. db.json 파일을 만듭니다. 
    {
    "topics": [
        {
            "id": 1,
            "title": "REST",
            "body": "REST is ..."
        }
    ],
    "comments": [
        {      
            "id": 1,
            "body": "POST is ...",
            "topicId":1 
        },
        {
            "id": 2,
            "body": "READ comments is ...",
            "topicId":1 
        }
    ]
}
  5. json-server 실행
    json-server --watch db.json
    => db.json 파일이 생성됩니다.
  6. 개발자 도구의 Network 탭을 열고 ESC 키를 눌러서 콘솔창도 열어주세요.
  7. 실행버튼을 누르면 자바스크립트 코드가 실행됩니다.
